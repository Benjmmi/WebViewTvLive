## 功能
- 自动更新频道列表
- 多直播源
- 上下键换台反转


## 获取应用
- GitHub 自己打包<br>
https://github.com/Benjmmi/AITVLive <br>


## 缺点
- 基于 WebView 构建，对设备性能有一定的要求，详见[设备要求](#设备要求)
- 载入频道稍长，需要等待一定时间
- 无法选择视频清晰度，帧率有限（大部分为 24FPS）
- 由于版权限制，官网可能不会播放某些时段的节目

## 常见问题（FAQ）
- 遇到问题自己解决，解决不了找 AI
- [查看 FAQ](FAQ.md)
